KinectFaceTraking2D ANE 
================================================

このコードは Windows 用の ANE　を作成する際のサンプルです。

AdobeサイトにあるString や Numberのみならず、VectorやRect型のデータ取得方法についても書かれています。
より実践的な内容になっています。
